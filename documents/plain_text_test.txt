HOW IT WORKS
============

1. File Detection
   LocalWise scans the documents/ folder recursively for all supported file types.
   It uses glob patterns to match file extensions.

2. Text Extraction
   For plain text files like this one, the content is read directly.
   Multiple text encodings are tried (UTF-8, Latin-1, CP1252, etc.) to
   ensure compatibility with different file sources.

3. Content Processing
   The extracted text is split into chunks using LangChain's
   RecursiveCharacterTextSplitter for optimal AI processing.

4. Vector Embeddings
   Text chunks are converted to vector embeddings using Ollama
   for semantic search capabilities.

Compatibility: Handles different text encodings automatically
Scale: Can process thousands of text files efficiently
